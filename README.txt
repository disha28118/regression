# House Price Prediction

## Description
This is a simple linear regression project to predict house prices based on features like Bedrooms, Bathrooms, Area, and Age.

## How to Run the Project

### 1. Set up your environment:
```bash
pip install -r requirements.txt
```

### 2. Run the main program:
```bash
python main.py
```

## Project Structure
- main.py: Entry point
- train_model.py: Trains and saves the regression model
- predict.py: Loads the model and makes predictions
- data/house_prices.csv: Sample dataset
- model/: Folder to store trained model
